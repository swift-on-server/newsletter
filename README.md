# Newsletter

Newsletter issues for the swiftonserver.com website

## Dependencies

The project uses [mjml](https://mjml.io/) to generate HTML newsletter issues.

`brew install mjml`

## HTML Generation

`mjml issue-XX.mjml > issue-XX.html`
